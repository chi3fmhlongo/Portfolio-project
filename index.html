About This Project

This project was inspired by my personal journey to improve my daily habits and boost productivity. After struggling to keep track of my tasks and habits using traditional methods, I decided to create a tool that would help me and others stay organized and motivated. The project started as a simple app but evolved into a comprehensive habit-tracking solution. This project is also a part of my portfolio for Holberton School.
